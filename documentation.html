Error Codes

Our API uses standard HTTP response codes to indicate the success or failure of an API request.

Code Status Meaning
200 Success The request was successful and the neural analysis is complete.
400 Bad Request The request was unacceptable, often due to missing a required parameter.
401 Unauthorized No valid API Key was provided in the authorization header.
429 Too Many Requests You have hit the rate limit for your current pricing plan.
500 Server Error Something went wrong on NexGen Core's end. We're on it!
